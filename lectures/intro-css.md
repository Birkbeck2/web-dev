# Introduction to CSS: Prep

## Learning outcomes

* What is CSS?
* Basic CSS syntax
* Selectors
* CSS integration
* CSS basic properties
* CSS Box Model
* CSS Margin, Border, Padding, Content
* CSS links, background-image, lists, tables, display

## Pre-sessional activities
